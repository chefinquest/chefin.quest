Study 004: Chromatic Lattice

The reference is all density and interference: tiny rotated blocks repeat like a textile, then the whole grid bends into electronic corridors. This sketch builds that effect by drawing each mark three times, as misregistered red, cyan, and pale ink.

Press S to save, R for a new seed. The density slider changes the spacing of the woven dash field, which makes the moiré either breathe or overload.

  • Black bars preserved above and below the image area.
  • Horizontal strata echo the reference’s upper, middle, and lower bands.
  • Red/cyan misregistration creates the vibrating 3D-print feeling.
  • Yellow appears only as rare sparks inside the densest corridors.

Process notes

I treated the image as a damaged RGB textile/circuit print. The main move is not a single filter — it is a field of small marks whose positions are domain-warped, clipped into bands, and then redrawn with chromatic offsets.

// Core trick: one woven dash becomes three misregistered ink passes.
const p = warpPoint(x, y);
const a = fieldAngle(p.x, p.y);
drawDash(p.x - split, p.y, a, redInk);
drawDash(p.x + split, p.y, a, cyanInk);
drawDash(p.x, p.y + 1, a, whiteInk);
