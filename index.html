Study 001: Garden Plotter

This is not a copy; it is a reconstruction of the likely method. The important idea is to draw in layers: paper → masks → fields → seams → accents → grain.

Press S to save, R for a new seed. Watch how changing density changes the algorithm’s character, not just the size of a shape.

How the reference was probably made

The piece looks like a generative print rather than a clean vector drawing. I’d reverse-engineer it as a p5.js sketch using lots of semi-transparent marks on a warm paper background. The visual clues:

  • Soft rectangular panels: four vertical-ish columns and horizontal breaks suggest clipping masks or a plotter/grid composition.
  • White seams: the bright gaps are not empty accidents; they are probably drawn or preserved as negative-space masks.
  • Pastel salmon dominance: one base color is used everywhere at low alpha, so density creates darker passages.
  • Green/yellow/orange islands: accent colors appear only where a second noise field crosses a threshold.
  • Diagonal pale bands: repeated slash-like voids suggest a stripe mask: points near a diagonal equation are skipped or covered with translucent paper.
  • Speckled texture: thousands of tiny dots/noisy pixels sit over the whole image, imitating risograph, inkjet, or aged paper.

1. Think in layers

Don’t start by drawing “a shape.” Start by deciding what each layer contributes.

  1. Paper: off-white background with subtle grain.
  2. Mask: a ragged rectangle plus vertical and horizontal seams.
  3. Base field: thousands of salmon/pink strokes following noise.
  4. Accent field: fewer green, yellow, orange, and pink clusters.
  5. Plotter lines: faint cyan/white seam lines and diagonal gaps.
  6. Damage/noise: stipples, faded edges, imperfect ink.

2. Use noise as decision-making

Perlin noise gives you coherent neighborhoods. A random point says “anything can happen anywhere.” A noise point says “things that are near each other are related.”

let n = noise(x * 0.006, y * 0.006);
if (n > 0.38) drawBasePigment(x, y);
if (noise(x * 0.018 + 50, y * 0.018) > 0.63) drawAccentIsland(x, y);

The first field uses a low frequency for broad cloudy areas. The accent field uses a higher frequency and threshold so color arrives in islands.

3. Color reasoning

The palette works because it is mostly warm and low-contrast, with only a few sharp notes.

  • Paper: creamy white, not pure white. Pure white would feel sterile.
  • Base: salmon, peach, dusty rose. These occupy most of the surface.
  • Botanical accents: yellow-green breaks the pink field without fighting it.
  • Hot accents: yellow and orange are sparse; they become focal points.
  • Cyan seams: very pale blue-green makes the grid feel luminous and aged.

4. Shape language

The reference is made of small marks, not big perfect shapes. Use short rotated rectangles, blobs, tiny lines, and points. Let density build form.

push();
translate(x, y);
rotate(noise(x*.01, y*.01) * TWO_PI);
rect(0, 0, random(3, 18), random(1, 8), 2);
pop();

That one pattern already creates a “printed brush stroke” if you draw it thousands of times with low alpha.

5. Minimal starter sketch

If you want to rebuild this from scratch, start with this tiny version and add one layer at a time:

function setup() {
  createCanvas(827, 1105);
  pixelDensity(1);
  noLoop();
  randomSeed(12);
  noiseSeed(12);
  background('#fffdf0');

  // base pigment field
  for (let i = 0; i < 9000; i++) {
    let x = random(width), y = random(height);
    let n = noise(x * 0.006, y * 0.006);
    if (n < 0.34) continue;

    noStroke();
    fill(216, 128, 112, 28); // salmon with low alpha
    push();
    translate(x, y);
    rotate(noise(x * .01, y * .01) * TWO_PI);
    rect(0, 0, random(4, 22), random(1, 9), 2);
    pop();
  }

  // accent islands
  let colors = ['#93ad49', '#f2df18', '#e9992f', '#e8a0bc'];
  for (let i = 0; i < 900; i++) {
    let x = random(width), y = random(height);
    let a = noise(x * 0.018 + 80, y * 0.018 - 20);
    if (a < 0.61) continue;
    fill(random(colors) + '88');
    noStroke();
    ellipse(x, y, random(4, 18), random(2, 12));
  }

  // pale plotter seams
  stroke(170, 220, 205, 115);
  strokeWeight(3);
  for (let x of [width*.28, width*.57, width*.83]) line(x, 25, x + random(-12,12), height-35);
  for (let y of [height*.30, height*.61, height*.86]) line(25, y, width-25, y + random(-12,12));

  // print grain
  loadPixels();
  for (let i = 0; i < pixels.length; i += 4) {
    let g = random(-18, 18);
    pixels[i] += g; pixels[i+1] += g; pixels[i+2] += g;
  }
  updatePixels();
}
