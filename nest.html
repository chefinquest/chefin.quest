Study 005: Coral Nest

Instead of copying the photo literally, this keeps the important structure: an elliptical woven bowl, a dark hollow, straw-like overlaps, wild protruding arcs, and a background with enough texture to make the nest feel found.

Press S to save, R for a new seed. Drag the sliders and the algorithm redraws a new reef-built version of the nest.

  • Layered arcs mimic dry grass, but the palette is coral, pearl, seafoam, and deep lagoon teal.
  • Caustic curves and reef speckles create the cool marine background texture.
  • The center is deliberately darker so the woven rim reads as a nest instead of a flat wreath.

Process notes

The nest is drawn as over a thousand hairline Bezier splines arranged around an ellipse. Each strand chooses a lane: outer rim, inner bowl, cross-weave, or escaped filament. Color and thickness vary per lane so the composition keeps the photo’s fragile straw energy while shifting into a reef palette.

// Core move: one elliptical nest strand with noisy control points.
const a0 = random(TWO_PI);
const a1 = a0 + random(-0.9, 0.9) + CONFIG.curl * 0.006;
const p0 = nestPoint(a0, random(0.72, 1.22));
const p3 = nestPoint(a1, random(0.72, 1.22));
bezier(p0.x, p0.y, c1.x, c1.y, c2.x, c2.y, p3.x, p3.y);
